# Reactivities

This is a social networking app for sharing events and activities with friends.
This application is built using:

1. Asp.Net Core
1. SqLite
1. SignalR
1. React
1. Mobx
1. Semantic UI
1. React Final Form
1. React Hooks

The client side application is built using [Create React App](https://github.com/facebook/create-react-app).

## Running the application

Before running the application you need to create a [Cloudinary] (https://cloudinary.com/) account and insert its API key and secret to the API/appsettings.json file.

You can run the application by running the commands:

# Running Backend:

`cd API; dotnet run`

# Running the frontend:

first run:

`yarn install`

and then:

`yarn start`
